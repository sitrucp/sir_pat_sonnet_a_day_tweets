@SirPatStew #ASonnetADay Dashboard

Sir Patrick Stewart @SirPatStew has been doing Shakespeare Sonnet readings from his home during COVID-19 lockdown and they are quite good!

I was compelled to get @SirPatStew's Twitter data and use it to create an interactive table with links to each tweet to allow everyone to easily watch them, as well as create a visualization of the tweets' likes and retweet counts to show popularity. Updated regularly. Enjoy!
